# jvosk

WAV/MP3 transcription using Vosk. 
Built with Swing.

<img width="1824" height="1424" alt="image" src="https://github.com/user-attachments/assets/83072e9f-bc86-4c8f-9e46-7fce1a2508ba" />



How to Build:

1. Install Maven, Java 17+ and ffmpeg (for MP3s)
2. Build package with `mvn clean package`
3. Run application with `mvn exec:java -Dexec.mainClass=atri.palaash.jvosk.App`
